Conversación

Se trata de contar una pequeña historia con personajes, que para sincronizarlos vamos a aprender el uso del envío y la recepción de mensajes.

Con este proyecto nos proponemos:

  • Conocer y usar los eventos de tipo mensajes para sincronizar personajes.
  • Repasar el uso de escenarios.

Se pueden usar recursos como el movimiento de personajes hasta que toquen un objeto, los cambios de disfraz para caminar o bailar, cambios de escenarios, etc.

Os dejamos un enlace al proyecto y el mismo embebido a continuación.
